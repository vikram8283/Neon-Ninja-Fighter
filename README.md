# ⚡ Neon Ninja Fighter

A 2D wave-survival action game built using Python and Pygame during Rapid Game Craft.

## 🎮 Gameplay

Fight against endless waves of enemies and survive as long as possible.

Players can switch between multiple weapons and battle different enemy types while earning points and progressing through increasingly difficult waves.

## ✨ Features

- Multiple weapons:
  - Sword
  - Bow
  - Gun

- Enemy types:
  - Brute
  - Archer
  - Gunner

- Progressive wave system

- Health and damage mechanics

- Score tracking

- Particle effects

- Menu, loading, wave-complete, and game-over screens

- Custom sprite and background assets

## 🛠️ Built With

- Python
- Pygame

## 🎯 Controls

| Key | Action |
|------|---------|
| A / D | Move |
| W / ↑ | Jump |
| Space | Attack |
| E | Switch Weapon |
| Q | Quit |
